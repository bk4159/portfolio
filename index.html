Projects

African Crafts & Jewellery

African Crafts and Jewellery is a website for a local jewellery vendor and is made using Vue.js. This will allow the jeweller to direct customers to their own website during festivals. I made the website prototype, including the Home page, the Products page (which gives images, names, and prices for products), and the Contact page.

Crowdlegal Dashboard

The Crowdlegal Dashboard is a web application made in a group capstone project for the organization CrowdLegal. The dashboard is able to visualize metrics and more for a CrowdLegal user. I helped with project management, the backend, and the frontend.

One backend component I created is the functionality to export data into CSV format. Unit tests were also made.

In the frontend, I made the frontend and backend logic which shows the campaign name and allows users to navigate between their campaigns. I enhanced the responsiveness and cross-browser functionality of the entire web app. I created the frontend and backend logic for a form that gets user feedback on the dashboard. Finally, for the header, title, tabs, and more, I made aesthetic changes throughout the dashboard to have a better UI/UX.

Patients Management Subsystem

Patients Management Subsystem is a web application for a hospital management system. Among other features, one can add, remove, and update patient or staff profiles.

This prototype was made in a group project where I mainly worked on the frontend using React. I made the frontend and backend logic for the Consult Patient File and the Visualize Division pages. The Consult Patient File refers to the database and shows patient information for the searched Patient ID, including the date of birth, phone numbers, addresses, and more. The Visualize Division page refers to the database and gives information on a given division. The Visualize Division has the functionality to discharge patients in the Admissions column and to admit patients in the Admission Requests column.

Rentals Database Interface

The Rentals Database Interface is a CLI for the database of a vacation rental application. The interface allows for a user to interact with a PostgreSQL database. The user's capabilities on the database depends on the user type (Admin, Host, Guest, etc.). For example, hosts can see and create listings whereas guests can only view the listings.

This prototype was made in a group project. I created the backend Java program that interacted with the database. The program welcomes the users, allows them to log into their accounts, and interact with the database depending on their user type.

NBA Stat Grapher

This prototype was made in a group project. I made the Welcome page, the tooltips, and I also worked to make the Graphing page look and feel better.

SEGEats
